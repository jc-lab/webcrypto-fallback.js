# webcrypto-fallback

# Usage

```bash
$ npm install --save webcrypto-fallback
```

```typescript
import 'webcrypto-fallback/dist/shim';
```

OR 

```typescript
import { shim } from 'webcrypto-fallback';
shim(self);
```
